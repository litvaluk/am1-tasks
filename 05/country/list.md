# Show countries

Show all countries.

**URL** : `/country`

**Method** : `GET`

## Success Response

**Code** : `200 OK`

**Content** : In this example, there are 3 countries present.

```json
[
  {
    "id": 1,
    "name": "USA"
  },
  {
    "id": 2,
    "name": "Cuba"
  },
  {
    "id": 3,
    "name": "Czech republic"
  }
]
```
